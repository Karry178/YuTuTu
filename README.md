# YuTuTu - 鱼图图

智能图片管理平台 - 基于 Vue 3 + Spring Boot 的全栈项目

## 📁 项目结构

```
YuTuTu/
├── Yu-picture-backend/     # 后端项目（Spring Boot）
│   ├── src/
│   ├── pom.xml
│   └── ...
├── yu-picture-frontend/    # 前端项目（Vue 3 + TypeScript）
│   ├── src/
│   ├── package.json
│   └── ...
└── README.md
```

## 🚀 技术栈

### 后端
- **框架**: Spring Boot 3.x
- **语言**: Java 17+
- **构建工具**: Maven

### 前端
- **框架**: Vue 3
- **语言**: TypeScript
- **构建工具**: Vite
- **路由**: Vue Router
- **状态管理**: Pinia
- **代码规范**: ESLint + Prettier

## 📦 快速开始

### 后端启动

```bash
cd Yu-picture-backend
mvn spring-boot:run
```

### 前端启动

```bash
cd yu-picture-frontend
npm install
npm run dev
```

## 📝 开发说明

- 后端默认端口：`8080`
- 前端默认端口：`5173`
- 已配置 CORS 跨域支持
